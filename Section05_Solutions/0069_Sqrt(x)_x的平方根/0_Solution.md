# 69_Sqrt(x)_x的平方根 (Easy)

## 解法一：(二分) $O(logx)$

- 二分出最大的 $y$，满足 $y^2 ≤ x$。则 $y$ 就是答案。
- $mid ^ 2 ≤ x$，就说明左边界可以往右移一点。

- 可以这样处理**溢出**问题：
  - $int \ mid = l + 1ll + r >> 1;$，$1ll$ 表示 $long \ long$ 类型的 $1$。
  - $mid*mid <= x$ 还会溢出，所以可以将 $mid$ 移到等号右侧 $mid <= x / mid$。

**时间复杂度分析**：二分的时间复杂度是 $O(logx)$。
