# 53_Maximum Subarray_最大子序和 (Easy)

## 解法一：动态规划

- 状态表示：$f(i)$ 表示所有以 $nums[i]$ 结尾的区间中的最大和是多少。

- 递推关系：$f(i) = max(nums[i], f(i-1) + nums[i]) = nums[i] + max(0, f(i-1))$。

  - 用变量 `last` 来记录 $f(i-1)$，得 $f(i) = nums[i] + max(0, last)$，就可以算出来 $f(i)$。之后再把 `last` (即 $f(i-1)$) 更新成 $f(i)$。
