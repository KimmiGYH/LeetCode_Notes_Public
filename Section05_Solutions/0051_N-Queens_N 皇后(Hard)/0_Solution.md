# 51. N-Queens_N 皇后(Hard)



##### (按行枚举) $O(n!)$

解释说明

对角线 `dg[u+i]`，反对角线 `udg[n−u+i]` 中的下标 `u + i` 和 `u - i + (n - 1)` 表示的是截距。

下面的 `(x, y)` 相当于`(u, i)`：

（1）反对角线 `y = x + b`, 截距 `b = y − x`，因为我们要把 `b` 当做数组下标，所以 `b` 不能是负的，所以我们 `+ (n - 1)`，保证是结果是正的
（2）而对角线 `y = −x + b`, 截距是 `b = y + x`，这里截距一定是正的，所以不需要加偏移量。
