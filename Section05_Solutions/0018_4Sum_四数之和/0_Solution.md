# 18_4Sum_四数之和

- 设置四个指针 $i$、$j$、$k$、$u$，且 $i < j < k < u$，固定 $i$、$j$

```
|-------- --------|-------|-------|
                  k       u
```

- $nums[k]↗ + nums[u]↘ + C \ge 0$
- 当 $nums[k]$ 上升时，nums[u]$ 必定时下降的
- 所以后两个指针 $j$、$k$ 可以从 $O(n^{2})$ 变成 $O(n)$
- 不重复的条件可以用当后一个元素$==$前一个元素 时，`continue` 跳过。
- 解决溢出问题可以将两项搬到运算符号右侧
