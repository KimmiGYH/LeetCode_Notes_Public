# 226. Invert Binary Tree_翻转二叉树 (Easy)



(递归，二叉树) $O(n)$

递归遍历整棵二叉树，遍历到每个节点时交换左右儿子。



##### 时间复杂度分析

每个节点仅被遍历一次，所以时间复杂度是 O(n)O(n)。
