# 485_Max Consecutive Ones_最大连续 1 的个数

### 解法：双指针

- 如果数字为 `0` 就 `continue` 跳过，直到数字为 `1`
- 引入指针 `j`，让 `j = i + 1;`
- 有连续的 `1` 时，将指针 `j` 不断右移直到边界
- 维持最长的连续 `1` 的个数
- 最后将指针 `i` 跳到 `j` 的位置，注意这里不用考虑 `i++` 带来的指针 `j` 回退问题，因为此题中 `nums[j]==0 ` 时要跳过，所以 `i++` 间接帮助完成这一步骤

**时间复杂度**

- 时间复杂度：$O(n)$
- 空间复杂度：$O(1)$
